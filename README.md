# Bug Tracker (MERN Stack)

## Overview

Bug Tracker is a full-stack MERN application that helps users manage software bugs efficiently.

## Features

* User Registration and Login
* JWT Authentication
* Create New Bugs
* View All Bugs
* Update Bug Status
* Delete Bugs
* Dashboard Statistics
* MongoDB Atlas Integration

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT Authentication

## Installation

Backend:

npm install
npm run dev

Frontend:

npm install
npm run dev
